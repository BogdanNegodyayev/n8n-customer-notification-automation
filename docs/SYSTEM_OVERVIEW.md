# System Overview

## Purpose

This workflow automates customer request processing and notification delivery.

The automation reduces repetitive manual work by connecting customer requests with internal communication channels.

---

## Components

- n8n
- Google Sheets
- Telegram Bot API
- Green API
- REST APIs
- Webhooks

---

## Workflow

Customer Request

↓

Google Sheets

↓

n8n

↓

Telegram Notification

↓

WhatsApp Notification
